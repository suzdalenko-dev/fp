XML
Ejemplo de XML: esta presentado por los canales de noticias, en este caso el Diario Montañes.
Hay cantidad de canales que ofrecen sus noticias en formato XML, por lo que es un formato muy utilizado en este tipo de aplicaciones.
Dentro de "EL Diario Montañes" podemos encontrar noticias de actualidad, deportes, cultura, etc. con los correspondientes XML
En mi caso he elegido el de la portada de noticias.

JSON
Ejemplo de JSON: es un formato de intercambio de datos ligero, fácil de leer y escribir para los humanos y fácil de analizar y generar para las máquinas.
En mi caso he eligido una API libre que proporciona datos de ejemplo en formato JSON.
Podemos traer un item determinado, todos los items, etc.
